For, While, and Do Loops

For loops are useful when we know beforehand how many times we would like the compiler to pass through the loop before exiting.

Example 1

for (int j=0; j<3; j++){
System.out.println(j);
}

In Example 1 we have told the compiler to go through the loop as long as j is less than 3. Therefore we know the compiler will go through the loop exactly three times and in this case will print a "0" on the first pass, a "1" on the second, and a "2" on the third. In this way we can control how many times the loop "goes around."


In contrast While loops are used when we are unsure how many times we would like the loop to execute. Instead we want the loop to continue going around while a condition is true. The statement following "while" will return a boolean value, and the loop will only execute if the statement returns true.


Example 2

int j=1;
int count=1;
while (j<1000000){
j=2*j;
count++;
}
System.out.println("It takes only " +count+ " doubles to reach a million." );

In the second Example we need to find out how many times we can double a quantity starting at one before we reach one million. On the first pass j will equal 2, 4 on the second, 8 on the third and so on. I used a while loop in this case because I do not know how many times my quantity needs to be doubled, and therefore do not know how many times I need the loop to execute.


A Do-While loop is like a While loop except that it evaluates a boolean expression at the end of the loop guaranteeing that the loop will execute at least the first pass.

do {
j=2*j;
count++;
}while (j<1000000);
System.out.println("It takes only " +count+ " doubles to reach a million." );
