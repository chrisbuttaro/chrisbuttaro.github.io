Math.random()

Math is a class of methods in Java for evaluating mathematical operations such as exponents, square roots, trigonometric functions, absolute value, and logarithms. One specific method in Math is the method called random. The random method returns doubles in the range [0.0, 1.0). In other words it can return any double including between 0.0 and 1.0, including 0.0 but less than 1.0 randomly.

Example 1

double rps;
 rps=Math.random();

 if ( rps < 0.3333333 )
 {
 System.out.println( "Rock" );
  }
 else if ( rps < 0.6666667 )
 {  System.out.println( "Paper" );
 }else{
 System.out.println( "Scissors" );
 }

In Example 1 we are calling the Math.random() method and storing which ever double it generates into the variable rps. In this case 1/3 of the time the system will print either "Rock," "Paper," or "Scissors." Math.random() can be manipulated to generate any range of numerical values when we use it in conjunction with operators.

Example 2

  a = 1 + (int)( 10*Math.random() );
 System.out.println(a);

 b = 1 + (int)( 100*Math.random() );
 System.out.println(b);

 c=5+ (int)(6*Math.random());
 System.out.println(c);

In Example 2 the system will print an integer between 1-10 on the first line, an integer between 1-100 on the second, and an integer between 5-10 on the third.
