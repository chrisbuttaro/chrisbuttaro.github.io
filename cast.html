Casting

Casting is used to define a variable as a more specified data type of the same hierarchy. We can cast a variable of type Number to be handled by the compiler as type Int because Int is a subclass of Number. However if we try to cast variable of type Int to be treated as a Number the compiler will through an exception. Casting automatically tells the compiler to preform a runtime check that our variable is defined accordingly if it is not then an exception will also be thrown.

Example 1

Number x=0;
x= (int)3.55;
System.out.println(x)// the system will print "3"

Example 2

Int x=0;
x=(Number)3.55// an exception will be thrown because we cannot cast up the hierarchy.

One useful example of casting is in using Math.random() to produce integers. By default Math.random() is defined to produce numbers in the range [0.0, 1.0), but by multipling the value produced by Math.random() and casting it as an integer we can tell the compiler to drop the digits after the decimal thus producing only integers.

Example 3

  a = 1 + (int)( 10*Math.random() );
 System.out.println(a);


In Example 3 the system will display an integer between 1-10.
