switch vs if/else

Switches should be used in place of an if/else statement if possible. Switches are well organized, easy to understand, and can speed up your program. Switches are used for testing a single condition such as an integer, enumerated value, or a String.

Example 1

 switch(singleCondition){
  case conditionValue: executeFunction1();
   break;
  case conditionValue: executeFunction2();
   break;
  }

Example 2

int days=0;
System.out.print( "Which month? (1-12)");

month = keyboard.nextInt();

 switch(month){
  case 9:
  case 4:
  case 6:
  case 11: days = 30;
   break;
  case 2: days = 28;
   break;
  default: days = 31;
  }
System.out.print(days+" days")

In the second example we can see that the switch is checking for the condition of an integer variable called "month." If the month equals 9,4,6, or 11 then days will equal 30, or else if the integer is equal to 2 the days will equal 28, or else the switch will default the value of days to 31. The wordiness of my previous sentence is meant to convey how an using an if/else statement in this case would have been slightly more complicated.


When testing for multiple conditions a switch cannot be used and we may defer to an if/else statement. This can be achieved with using operators like || "or" and && "and."

Example 3

  if (condtionA || conditionB && conditionC){
  excuteFunction1();
 }else{
   excuteFuntion2();
  }

In Example 3 function 1 will execute in the case that condition A is true or if both conditions B and C are true. Otherwise function 2 will execute.

Example 4

int age =20;
boolean onGuestList = false;
double allure =7.5;
String gender = "M";

if ( onGuestList || age >= 21 || (gender.equals("F") && allure >=8)){
  System.out.println("You are allowed to enter the club.");
}else{
  System.out.println("You are not allowed to enter the club.");
}

Example 4's program displays "You are allowed to enter the club." If any one of these three conditions apply:

  • 1. If onGuesList equals true.
  • 2. If age is greater than or equal to 21.
  • 3. If the string "gender" is equal to "F" and allure is also equal to 8 or greater.

If neither 1,2,or 3 are completely true then the program will display "You are not allowed to enter the club."
